Persian interval Datepicker based on Amin Rahimi's Persian Datepicker

persian-interval-datepicker attributes

  • display (Defaults: inline) : could be inline or multiline for displaying Datepickers

  • format (Defaults: yyyy/MM/dd) format of displaying date in Datepickers

  • start-date-model (Defaults: startDate) name of the startdate of interval, ng-model of datapicker consist of combination of startDate and endDate

  • end-date-model (Defaults: endDate) name of the enddate of interval, ng-model of datapicker consist of combination of startDate and endDate

  • start-date-label if defined, label with this attribute value will be placed before startDate Datepicker

  • end-date-label if defined, label with this attribute value will be placed before endDate Datepicker

  • start-date-placeholder if defined, placeholder attribute of startDate Datepicker will be set with this value

  • end-date-placeholder if defined, placeholder attribute of endDate Datepicker will be set with this value

  • range-min (Defaults:1) minimum of ranges between Datepickers dates.

  • range-max maximum of ranges between Datepickers dates
